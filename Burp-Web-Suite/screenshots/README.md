# Screenshots

These are **placeholder images**. Replace each one with a real screenshot from
Burp Suite Professional, **keeping the exact filename** — the main README
references them by name, so once you overwrite a placeholder the image appears
automatically (no README edit needed).

## How to replace

1. Load `web-suite.bapp` in Burp (Extensions → Add → Extension type: Java).
2. Open the **Web-Suite** tab, go to the module you want to capture.
3. Screenshot that panel (Windows: `Win`+`Shift`+`S`), ideally ~960×540.
4. Save it over the matching file below, same name, in this folder.
5. Commit — GitHub renders it in the README.
